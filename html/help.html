Notes


DDC project files using IIR filter types other than 'Peaking' generated by this version are not compatible with Viper's Toolbox 2.0.

This is because these IIR filter types are not supported by the original version of Viper's Toolbox.

Of course you can still generate *.vdcprj files that are compatible with this older version. To do that, just make sure that all filter types are set the 'Peaking' or use the option called 'Project File v1.0' in the 'Export' submenu. Keep in mind that if you use the export option, the filter types of all calibration points are set to 'Peaking' by force.


The VDC importer is not guarenteed to support VDCs, that have been built using IIR filters other than 'Peaking' very well.
