Key Shortcuts


Project:

Ctrl-O: Open project file

Ctrl-S: Save project file

Ctrl-X: Close project


Edit:

Ctrl-N: Add point

Ctrl-R: Remove selected point(s)

Ctrl-Z: Undo

Ctrl-Shift-Z: Redo

Alt-Z: View undo history

F2: Perform stability check on all filters


View:

Ctrl-H: Hide graph

Ctrl-Print: Take screenshot of graph


Selection:

Ctrl-Shift-I: Invert gain

Ctrl-Shift-F: Shift frequencies by n-Hz


Help:

F1: Shortcuts


Notes:

- Double-click a cell to edit it

- Right-click one of the plots to enable movement/zoom
